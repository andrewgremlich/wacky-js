A JS History Lesson
  • Netscape was the internet company of the 90's.
  • Netscape wanted interactivity and dynamic.
  • Netscape evaluated languages for web.
  • Java was considered with applets.
  • Netscape hired Brendan Eich to embed a Scheme language into Navigator.
  • Eich saw the advantage of a dynamic language.
  • Instead of Scheme, he made Mocha in 10 days.
  • Mocha drew syntax inspiration from Java.
  • Mocha was accepted and later named LiveScript
  • Several months later LiveScript renamed to JavaScript.
    • Attract Java Developers.
    • No similarities to Java.
  • Microsoft also wanted a language for the web.
  • They made JScript for IE.
  • ECMAScript was established and Microsoft would further diverge from the standard among versions.
ECMAScript Version History.
  • ES4
    • so many new features
    • Reps from Yahoo and Microsoft expressed concerns for complex language.
    • ES4 was blocked. ES3.1 had minor upgrades.
    • ES4 features would come later in ES6.
    • Adobe picked up ES4 in ActionScript in Adobe Flash.
    • Adobe could never drive ES features.
  • Microsoft took browser share from Netscape and dictated how web went.
  • Standarization wasn't prevalent.
  • Different JS programs to support different browsers.
  • ES3.1 was continually worked on, and eventually renamed to ES5 by 2009.
  • Around the same time of 2009, Google Chrome and Firefox started to gain traction.
  • Chrome and Firefox picked up Microsoft slack with IE. They took users from IE.
  • Chrome and Firefox had more of a say in Web Standards than Microsoft.
  • ES6 released in 2015 (hello es4!)
  • A release every year afterwards.
